## 介绍

Amall项目是一个B2C模式的电商网站, 前端 UI 原型是参考蘑菇街商城界面进行设计, 

以react框架为技术骨架来开发.

## 用到的技术

- react + react-hooks
- UI   antd
- 状态管理 react-redux
- 路由使用 react-router, react-router-dom
- 异步使用 react-saga
- 请求使用 axios
- 辅助工具 ahooks

## 后端项目地址
